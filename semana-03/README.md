# Semana 03: Encapsulamiento y Constructores

## Descripción

Esta semana se implementa el encapsulamiento mediante modificadores de acceso (private, public) y se crean getters y setters. También se trabaja con sobrecarga de constructores y validaciones.

## Estructura de Archivos

```
semana-03/
├── README.md (este archivo)
├── src/
│   ├── Student.java
│   ├── Instructor.java
│   ├── Pool.java
│   ├── Schedule.java
│   ├── AquaticsCenter.java
│   └── Main.java
└── docs/
    └── MEJORAS.md
```

## Objetivos de Aprendizaje

- Aplicar encapsulamiento con modificadores de acceso
- Crear getters y setters
- Implementar sobrecarga de constructores
- Agregar validaciones en setters

## Compilación y Ejecución

```bash
# Compilar
javac -encoding UTF-8 src/*.java

# Ejecutar
java -cp src Main
```

## Conceptos Clave

- Modificadores de acceso (private, public)
- Encapsulamiento
- Getters y Setters
- Sobrecarga de constructores
- Validaciones
