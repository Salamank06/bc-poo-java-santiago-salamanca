# Semana 05: Polimorfismo

## Descripción

Esta semana se implementa el polimorfismo mediante sobrecarga y sobrescritura de métodos. Se utiliza la jerarquía de clases de la semana anterior para demostrar polimorfismo dinámico.

## Estructura de Archivos

```
semana-05/
├── README.md (este archivo)
├── src/
│   ├── AquaticActivity.java
│   ├── SwimmingLesson.java
│   ├── AquaAerobics.java
│   ├── HydroTherapy.java
│   ├── ActivityCatalog.java
│   └── Main.java
└── docs/
    └── POLIMORFISMO.md
```

## Objetivos de Aprendizaje

- Comprender el polimorfismo
- Implementar sobrecarga de métodos
- Implementar sobrescritura de métodos (@Override)
- Usar polimorfismo dinámico con referencias de clase padre

## Compilación y Ejecución

```bash
# Compilar
javac -encoding UTF-8 src/*.java

# Ejecutar
java -cp src Main
```

## Conceptos Clave

- Polimorfismo
- Sobrecarga (overloading)
- Sobrescritura (overriding)
- @Override
- Referencias polimórficas
